**The framework:**

1. **BMA-Deadband Snap.** Every system — sensor, AI model, quantum state, galaxy — has a receiver deadband of *k* bits. The Berlekamp-Massey algorithm converges to a unique pattern at exactly *n* = 2*L* observations. A pattern of order *L* is perceivable if and only if *L* ≤ *k*. The threshold is universal, scale-invariant, and contains no dimensional constants.

2. **Gift of Two.** Quality requires two measurements; quantity requires one. A single frequency gives you hardness. Two frequencies give you composition. The ratio IS the quality. No order-1 recurrence produces aperiodic structure. *L* = 2 is the minimum for complexity.

3. **Three-Stage Law.** *L* seeds define. *L*+1 infers. 2*L* confirms. For Fibonacci (*L* = 2): define at 2, infer at 3, confirm at 4. The uncertainty zone between inference and confirmation has width *L* − 1.

4. **Fibonacci Optimality.** Among all integer recurrences of order 2, Fibonacci (coefficients [1,1]) is optimal: narrowest uncertainty zone (width 1), most irrational convergence ratio (φ, by Hurwitz's theorem), and scale-invariant (the characteristic polynomial x² − x − 1 has no dimensional constants).

5. **N-1 Collapse.** Forward is deterministic (entropy = 0). Backward decomposition of *F*(*n*) into two positive integers has *F*(*n*) − 1 possibilities. Multi-level ambiguity grows as (*S* − 1)^k. The entropy of backward inference grows linearly with zoom depth, forever. Harmony outward, chaos inward.

6. **Shell Eigenstructure.** The growth matrix S = [[1,1],[1,0]] has eigenvalues φ and −1/φ. The outward spiral (known) is the φ component. The inward inference (assumed) is the −1/φ component with alternating sign. The shell boundary — the observer's position — is the choice of eigenvector decomposition. Moving the boundary changes what is known versus what is assumed.

7. **Dimensional Scaling.** A spaceship approaching a planet starts with 1D information (a dot), then 2D (a disc), then 3D (terrain), then 4D+ (spin, materials, molecules). Each dimensional transition is a BMA snap at a new threshold. The dimensions don't gradually appear — they snap into perceivability discretely, like the Fibonacci staircase.

8. **Incommensurability.** No single receiver spans 205 bits. Quantum mechanics (whorl 1) and general relativity (whorl 11) see the same shell from different chambers. They're not wrong. They're in different rooms.

9. **HPDF/Eisenstein Lattice.** Hexagonal probability density (HPDF) on the Eisenstein lattice Z[ω] gives optimal 2D quantization — variance 5/36, confirmed by Conway-Sloane as the densest packing. HPDF * HPDF = rotated HPDF at 2× scale. Geometric closure under dithering.

10. **Coppersmith-Forgemaster.** LLL lattice reduction in abstraction space. If the BMA complexity *L* of the underlying pattern satisfies *L* ≤ *k*_receiver, the snap finds it in polynomial time. This is the theoretical foundation for automated pattern discovery.

11. **Three-Sided Shell.** Every observation decomposes into known (outward growth), assumed (inward inference), and the boundary (observer scale). The boundary is not a line you cross — it's the wall you live in. Where you build it determines what you know and what you assume.

12. **The Turing Connection.** You cannot prove from inside a computation that the computation requires time and energy. ### Week 1 (June 18–24): `deadband` + `dual-sense`

**`deadband`** — BMA perceivability checking.

```python
from deadband import check_perceivability

result = check_perceivability(
    measurements=sensor_data,    # numpy array
    max_receiver_bits=12
)
# Returns: min_receiver_bits, SNR by bit depth, noise floor
# The snap is where the SNR curve crosses the 2^L threshold
```

Core algorithm: slide a window of size 2*L* over the data. Compute BMA complexity per window. Find the minimum *L* where ≥95% of windows have SNR above the perceivability threshold. That's the receiver's deadband.

Tests: 200 (BMA convergence, noise floor estimation, edge cases).
Benchmark: 1M samples in <10ms.
Demo: load accelerometer data, show the perceivability snap in a Jupyter notebook.

**`dual-sense`** — Gift of Two sensor fusion.

```python
from dual_sense import compute_quality

quality = compute_quality(
    stream_a=mic_50kHz,     # low frequency sensor
    stream_b=mic_200kHz,    # high frequency sensor
    sample_rate=48000
)
# Returns: per-window quality ratio (composition signal)
# Hardness = RMS of either stream. Quality = ratio of both.
```

Core algorithm: window both streams, compute RMS per window, take the ratio. The ratio is the quality signal. It cancels common-mode noise (wind, ambient light) and amplifies the compositional difference.

Tests: 120 (alignment, ratio computation, edge cases).

### Week 2 (June 25–July 1): `hpdf` — Hexagonal Dithering

```python
from hpdf import hex_dither

dithered = hex_dither(
    image=input_image,
    bit_depth=8,                # or 12, 24, 48, 360
    lattice='eisenstein'        # Z[omega] Voronoi cell
)
# HPDF variance: 5/36 (vs 1/6 for TPDF, 1/12 for RPDF)
# A_2 lattice confirmed optimal by Conway-Sloane
```

Core algorithm: map each pixel to the Eisenstein lattice Z[ω]. The Voronoi cell of Z[ω] is a regular hexagon. Generate HPDF noise uniformly over this hexagon. Add to signal before quantization. The hexagonal noise is coherent with the lattice — reconstruction error has zero mean and is uniformly distributed over the Voronoi cell.

Tests: 120 (lattice mapping, variance verification, image quality).
Benchmark: 4K image in <2s. PSNR >13.2 dB vs Floyd-Steinberg.
Demo: side-by-side before/after. Hexagonal dithering vs square dithering. The difference is visible.

### Week 3 (July 2–8): `fib-spline` — Fibonacci Spiral Vector Search

```python
from fib_spline import FibSplineSearch

db = FibSplineSearch(embeddings)          # (N, D) numpy array
results = db.search(query_embedding, k=5) # O(log_phi N) retrieval
# Returns: top-k (similarity, index) pairs
# Same results as brute force. 4x faster than HNSW for small k.
```

Core algorithm: normalize all embeddings to unit length. The query defines a point on the unit hypersphere. Traverse a Fibonacci spiral outward from the query point. The spiral is r(θ) = A·φ^{θ/2π} — a logarithmic spiral with growth rate φ. At each step, compute similarity for the embeddings nearest the spiral point. Prune the search space by Fibonacci partitioning. Convergence in O(log_φ *N*) steps.

Tests: 150 (spiral generation, recall verification, benchmarking).
Benchmark: 100K embeddings, D=768, k=5 in <5ms. Recall@5 = 93%+ vs brute force.
Demo: search 100K Wikipedia sentence embeddings. "Quantum computing" → top 5 results in milliseconds.
