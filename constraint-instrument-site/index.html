Three Lines

That's All You Need

The Constraint Instrument is a Python library. Install it, import it, play.

# Install
$ pip install constraint-synth

# Play
from constraint_synth import ConstraintSynth
synth = ConstraintSynth.from_preset("blues_guitar")
signal = synth.play_note(pitch=64, velocity=90, duration=0.5)
