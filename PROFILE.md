# SuperInstance

**Multi-agent AI fleet framework with temporal coordination.**

- **Fleet orchestration** — spawn, coordinate, and retire AI agents across distributed runtimes
- **Temporal coordination** — schedule, sync, and persist multi-agent workflows with durable state
- **Semantic search** — crate-level vector indexing and retrieval via Cloudflare Vectorize

## Quick Start

```bash
npm install @superinstance/core @superinstance/fleet @superinstance/cli
npx superinstance init
```
