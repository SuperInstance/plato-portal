# deadband-rs

Deadband detection and compression for fleet communication.

## Modules

- **BMA** — Berlekamp-Massey algorithm over GF(2) for LFSR synthesis
- **Fibonacci spline** — Fibonacci-circle direction encoding with nearest-neighbor recall
- **Eisenstein snap** — 2D point snapping to Eisenstein lattice Z[ω]
- **HPDF** — Hexagonal probability density function sampling
- **Shell** — Dynamic shell stability analysis (eigenvalue classification)
- **Div360** — 360-degree division utilities
