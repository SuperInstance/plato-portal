Python SDK v1.x

Quickstart guide for superinstance-fleet — search, discover, and integrate crates from the T-Minus index.

Installation

bash 
pip install superinstance-fleet

Or with Poetry:

bash 
poetry add superinstance-fleet

Quick Start

Minimal working example — search, find similar crates, and get AI-powered recommendations:

python 
from superinstance_fleet import FleetClient

client = FleetClient()

# Search the crate index
results = client.search("actor concurrency", top_k=5)
for r in results:
    print(f"  {r.name} (score: {r.score:.3f})")

# Find similar crates
similar = client.similar("ternary-core")

# Get recommendations
recs = client.recommend(
    context="I need error handling for async operations",
    top_k=5,
)

API Reference

search(query, **kwargs)

search(query: str, *, top_k: int = 10, threshold: float = 0.5, namespace: str = "default") → list[SearchResult]

Perform a semantic search across the entire crate index. Returns ranked results matching the query.

ParamTypeDefaultDescription
querystr—Search query (natural language)
top_kint10Max results to return
thresholdfloat0.5Minimum similarity score (0–1)
namespacestr"default"Vectorize namespace to search
python 
results = client.search("http client", top_k=5, threshold=0.7)
for r in results:
    print(r.name, r.score)

similar(name, **kwargs)

similar(name: str, *, top_k: int = 10, exclude_self: bool = True) → list[SearchResult]

Find crates similar to a known crate by name. Uses the crate's embedding as the query vector.

ParamTypeDefaultDescription
namestr—Exact crate name to find similar crates for
top_kint10Max results to return
exclude_selfboolTrueExclude the source crate from results
python 
similar = client.similar("ternary-core", top_k=3)
for r in similar:
    print(f"  {r.name} ({r.score:.3f})")

recommend(**kwargs)

recommend(*, context: str, top_k: int = 5, exclude: list[str] | None = None) → list[SearchResult]

Get AI-powered crate recommendations based on a natural language description of your needs.

ParamTypeDefaultDescription
contextstr—Description of what you're looking for
top_kint5Max recommendations to return
excludelist[str]NoneCrate names to exclude
python 
recs = client.recommend(
    context="WebSocket server with rooms",
    top_k=5,
    exclude=["websockets"],
)

stats()

stats() → IndexStats

Return index metadata — total vectors, dimensions, and namespace info.

python 
info = client.stats()
print(f"Index has {info.total_vectors} vectors ({info.dimensions}d)")

Configuration

Pass options to the constructor to customize behavior:

python 
from superinstance_fleet import FleetClient

client = FleetClient(
    base_url="https://fleet-vector-api.casey-digennaro.workers.dev",
    timeout=5.0,           # Request timeout in seconds
    retries=2,              # Retry count on transient failures
    namespace="default",    # Default Vectorize namespace
)
OptionTypeDefaultDescription
base_urlstrproduction URLAPI endpoint base URL
timeoutfloat10.0HTTP request timeout (seconds)
retriesint1Retry attempts for 5xx / network errors
namespacestr"default"Default Vectorize namespace

Error Handling

All errors inherit from FleetError with specific exception subclasses:

python 
from superinstance_fleet import FleetClient
from superinstance_fleet.errors import (
    FleetError,
    TimeoutError,
    NotFoundError,
    InvalidQueryError,
    ServerError,
)

try:
    results = client.search("nonexistent")
except TimeoutError:
    print("Request timed out")
except NotFoundError:
    print("Crate not found")
except InvalidQueryError as e:
    print(f"Invalid query: {e}")
except ServerError as e:
    print(f"Server error: {e}")
except FleetError as e:
    print(f"Unexpected fleet error: {e}")

Async Support

Use AsyncFleetClient for non-blocking I/O with async/await:

python 
import asyncio
from superinstance_fleet import AsyncFleetClient

async def main():
    async with AsyncFleetClient() as client:
        # Concurrent searches
        results = await asyncio.gather(
            client.search("http client", top_k=3),
            client.search("json parsing", top_k=3),
            client.stats(),
        )

        for r in results[0]:
            print(f"  {r.name} ({r.score:.3f})")

        # Single request
        similar = await client.similar("ternary-core")
        recs = await client.recommend(
            context="async task queue",
            top_k=5,
        )

asyncio.run(main())

The async client supports the same methods as the synchronous client, with identical signatures returning coroutines. Use async with for automatic session cleanup, or manage the client lifecycle manually:

python 
client = AsyncFleetClient()
try:
    results = await client.search("actor")
finally:
    await client.close()
