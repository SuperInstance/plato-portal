Node.js SDK v1.x

Quickstart guide for @superinstance/tminus-client — search, discover, and integrate crates from the T-Minus index.

Installation

bash 
npm install @superinstance/tminus-client

Quick Start

Minimal working example — search, find similar crates, and get AI-powered recommendations:

js 
import { TMinusClient } from '@superinstance/tminus-client';

const client = new TMinusClient();

// Search the crate index
const results = await client.search('actor concurrency');
console.log(`Found ${results.length} crates`);

// Get similar crates
const similar = await client.similar('ternary-core');

// Get recommendations
const recs = await client.recommend({
  context: 'I need error handling for async operations',
  topK: 5
});

API Reference

search(query, options?)

search(query: string, options?: SearchOptions): Promise<SearchResult[]>

Perform a semantic search across the entire crate index. Returns ranked results matching the query.

ParamTypeDefaultDescription
querystring—Search query (natural language)
options.topKnumber10Max results to return
options.thresholdnumber0.5Minimum similarity score (0–1)
options.namespacestring'default'Vectorize namespace to search
js 
const results = await client.search('http client', {
  topK: 5,
  threshold: 0.7
});
results.forEach(r => console.log(r.name, r.score));

similar(name, options?)

similar(name: string, options?: SimilarOptions): Promise<SearchResult[]>

Find crates similar to a known crate by name. Uses the crate's embedding as the query vector.

ParamTypeDefaultDescription
namestring—Exact crate name to find similar crates for
options.topKnumber10Max results to return
options.excludeSelfbooleantrueExclude the source crate from results
js 
const similar = await client.similar('ternary-core', { topK: 3 });
similar.forEach(r => console.log(`${r.name} (${r.score})`));

recommend(options)

recommend(options: RecommendOptions): Promise<SearchResult[]>

Get AI-powered crate recommendations based on a natural language description of your needs.

ParamTypeDefaultDescription
options.contextstring—Description of what you're looking for
options.topKnumber5Max recommendations to return
options.excludestring[][]Crate names to exclude
js 
const recs = await client.recommend({
  context: 'WebSocket server with rooms',
  topK: 5,
  exclude: ['ws']
});

stats()

stats(): Promise<IndexStats>

Return index metadata — total vectors, dimensions, and namespace info.

js 
const { totalVectors, dimensions } = await client.stats();
console.log(`Index has ${totalVectors} vectors (${dimensions}d)`);

Configuration

Pass options to the constructor to customize behavior:

js 
const client = new TMinusClient({
  baseUrl: 'https://fleet-vector-api.casey-digennaro.workers.dev',
  timeout: 5000,       // Request timeout in ms
  retries: 2,            // Retry count on transient failures
  namespace: 'default', // Default Vectorize namespace
});
OptionTypeDefaultDescription
baseUrlstringproduction URLAPI endpoint base URL
timeoutnumber10000HTTP request timeout (ms)
retriesnumber1Retry attempts for 5xx / network errors
namespacestring'default'Default Vectorize namespace

Error Handling

All errors are instances of TMinusError with a machine-readable code:

js 
import { TMinusClient, TMinusError } from '@superinstance/tminus-client';

try {
  const results = await client.search('nonexistent');
} catch (err) {
  if (err instanceof TMinusError) {
    switch (err.code) {
      case 'TIMEOUT':
        console.error('Request timed out');
        break;
      case 'NOT_FOUND':
        console.error('Crate not found');
        break;
      case 'INVALID_QUERY':
        console.error('Invalid search query');
        break;
      case 'SERVER_ERROR':
        console.error('Server error:', err.message);
        break;
      default:
        console.error('Unknown error:', err.code);
    }
  } else {
    throw err;
  }
}

CLI Usage

The package ships a tminus CLI for quick lookups from the terminal:

bash 
# Search
npx tminus search "actor concurrency"

# Find similar crates
npx tminus similar ternary-core

# Get recommendations
npx tminus recommend --context "WebSocket server with rooms"

# Index stats
npx tminus stats

TypeScript Types

Key interfaces exported from the package:

typescript 
interface TMinusClientOptions {
  baseUrl?: string;
  timeout?: number;
  retries?: number;
  namespace?: string;
}

interface SearchOptions {
  topK?: number;
  threshold?: number;
  namespace?: string;
}

interface SimilarOptions {
  topK?: number;
  excludeSelf?: boolean;
}

interface RecommendOptions {
  context: string;
  topK?: number;
  exclude?: string[];
}

interface SearchResult {
  name: string;
  score: number;
  description?: string;
  version?: string;
  metadata?: Record<string, unknown>;
}

interface IndexStats {
  totalVectors: number;
  dimensions: number;
  namespaces: Record<string, { vectorCount: number }>;
}

class TMinusError extends Error {
  code: 'TIMEOUT' | 'NOT_FOUND' | 'INVALID_QUERY' | 'SERVER_ERROR';
}
