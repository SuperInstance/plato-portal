## The Dojo Concept

Each *log.ai domain is a verification arena — the "show me" after "I know kung fu."

```
Forge (build skill) → Dojo (verify skill) → Equipment (deploy skill)
     ↑                       ↓
  captain's log         *log.ai frontend
  (claims mastery)      (proves mastery)
```

The dojo doesn't just test if a skill works — it tests if the agent:
- Knows WHEN to use it
- Knows when NOT to use it
- Knows how it would react under pressure
- Can step back and decide about itself for the task at hand

This is meta-cognition at the application layer. The agent knows-itself.

## The Edge-Ware Connection

Edge-ware + git-agent standard = any skill on demand.

An edge device running a mature git-agent doesn't need every skill pre-installed.
It clones the skill repo, runs the bootcamp, and becomes the specialist.

The "at a thought" speed:
1. Agent encounters novel situation
2. Recognizes skill gap (meta-cognition)
3. Searches fleet skill registry
4. Clones relevant skill repo
5. Runs bootcamp (seconds on edge, milliseconds for cached)
6. Applies skill to task
7. Reports outcome for fleet learning
