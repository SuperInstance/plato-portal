💻 Quick Examples

Python SDK — Create an agent with persistent memory and spectral fleet coordination. 
from superinstance.agent import Agent, AgentConfig
from superinstance.fleet import Fleet

agent = Agent(AgentConfig(name="researcher", model="gpt-4"))
agent.remember("User prefers concise summaries")

fleet = Fleet(name="analysis-team")
fleet.add_agent(agent)
result = fleet.dispatch("Analyze this dataset")
print(result)
Node.js SDK — Two agents coordinating through the t-minus dispatcher. 
import { TminusClient } from "@superinstance/tminus-client";

const client = new TminusClient("ws://localhost:8765");
await client.connect();
await client.register("alpha", { timbre: "direct" });
await client.subscribe("hello");
await client.cue("beta", 0, "hello", { task: "greet" });

🧭 How It Works

⏱️
T-Minus Protocol
Temporal cueing with offsets. Negative offsets give head starts, positive create countdowns. Zero is immediate.
🛳️
Fleet & Vessels
Groups of agents coordinated through a central dispatcher with spectral load balancing.
🍶
Bottles
Encapsulated message containers that preserve state across agent handoffs.
⚖️
Conservation Law
Total fleet resources are conserved. Spectral eigenvalues track balance and trigger rebalancing.
