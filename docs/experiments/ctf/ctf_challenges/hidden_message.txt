# Hidden Message — Consonance-to-ASCII Challenge
#
# A WAV file (hidden_message.wav) contains a melody.
# Each note was synthesized at a specific position in consonance space.
# The consonance score of each note encodes one character of the message.
#
# Encoding: ASCII_code = round(consonance_score * 95) + 32
# This maps scores (0.0 - 1.0) to printable ASCII (32 - 127).
#
# To solve:
# 1. Analyze the WAV file to determine each note's fundamental frequency
# 2. Reverse-engineer the lattice position from the frequency
# 3. Compute the consonance score for each position
# 4. Convert scores to ASCII characters
#
# OR: Use the dial interpreter's --log feature to get consonance scores directly.
#
# Hint: The melody visits 18 positions. The message is a quote about music.
#
# The consonance scores (for verification):
# 0.863, 0.947, 0.863, 0.905, 0.337, 0.863, 0.905, 0.337,
# 0.947, 0.863, 0.863, 0.905, 0.337, 0.905, 0.863, 0.905,
